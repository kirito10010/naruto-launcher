# 火影忍者Online启动器

一款基于 Electron 框架开发的火影忍者Online页游启动器，支持游戏多开、速度调节、主题切换等功能。

## ✨ 功能特点

- 🎮 **游戏多开**：支持多个游戏窗口同时运行，每个窗口独立登录不同账号
- ⚡ **速度调节**：支持1x、2x、4x、6x、10x、20x多档速度调节
- 🎨 **主题切换**：提供明亮、暗黑、深海、火影四种主题
- 🔄 **自动更新**：支持自动检测并更新到最新版本
- 💾 **缓存管理**：一键清空缓存功能
- 💬 **QQ群**：加入官方QQ群获取更多帮助

## 🚀 快速开始

### 开发环境

```bash
# 安装依赖
npm install

# 启动开发模式
npm start
```

### 打包发布

```bash
# 打包Windows版本
npm run build
```

打包输出目录：`release-vX.X.X/`

## 🎯 使用说明

1. **启动游戏**：点击启动器中的"火影忍者Online"图标即可开始游戏
2. **多开游戏**：在游戏窗口点击"多开"按钮创建新窗口
3. **调节速度**：使用游戏窗口顶部的速度下拉菜单调节游戏速度
4. **切换主题**：在启动器点击右上角的主题按钮切换主题

## ⚠️ 注意事项

1. **Windows 11 用户**：请确保关闭"内存完整性"功能，否则可能影响速度调节功能
2. **缓存清理**：清空缓存会清除所有登录状态，请谨慎操作
3. **网络要求**：建议使用高速网络下载更新
